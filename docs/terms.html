1サービス内容

Twin Mirror（以下「本アプリ」）は、ユーザーが提供する2名の成人の顔写真を入力として、AI 画像生成モデルにより仮想的な「子ども（0〜20歳）」のイメージを生成するエンターテインメントアプリです。

2利用条件

  • 本アプリに入力できる写真は、18歳以上の方の写真に限ります。
  • 写真の被写体本人の同意を得た上でご利用ください。
  • 第三者の肖像権、プライバシー権、その他の権利を侵害する形で本アプリを利用してはなりません。
  • 未成年の写真、性的・暴力的・差別的な内容を含む写真は入力しないでください。

3生成画像の取り扱い

本アプリで生成される画像はフィクションであり、生物学的・遺伝的に正確な予測ではありません。生成画像を他者へ提示する際は、AI 生成物であることを明示してください。生成画像の二次利用についてはユーザーの責任において行うものとします。

4データの取り扱い

入力された写真は、生成処理のために Google LLC が提供する Gemini API へ送信されます。送信前に Apple App Review Guideline 5.1.2(i) に基づく明示的な同意をユーザーから取得します。本アプリ自身は写真を端末外に長期保存しません。Google のプライバシーポリシーが適用されます。詳細はプライバシーポリシーをご確認ください。

5利用回数の制限

コスト管理のため、本アプリは1日あたりの生成回数を制限しています。本制限は予告なく変更される場合があります。

6免責

本アプリは現状有姿で提供されます。生成結果の品質、特定の用途への適合性、利用により生じた損害について、開発者は一切の責任を負いません。

7規約の変更

本規約は予告なく変更される場合があります。最新の規約は常に本ページに掲載されます。
