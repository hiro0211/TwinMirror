Twin Mirror 利用規約

最終更新日：2026年5月17日

重要：本アプリで生成される画像は、AIによる娯楽目的のフィクションであり、実際の遺伝的予測ではありません。生成された画像が将来の実在の子供の容姿を予測・保証するものではありません。

第1条 サービス内容

Twin Mirror（以下「本アプリ」）は、ユーザーが提供する2名の成人の顔写真を入力として、AI画像生成モデルにより仮想的な「赤ちゃん」のイメージを生成するエンターテインメントアプリです。

第2条 利用条件

  • 本アプリに入力できる写真は、18歳以上の方の写真に限ります。
  • 写真の被写体本人の同意を得た上でご利用ください。
  • 第三者の肖像権、プライバシー権、その他の権利を侵害する形で本アプリを利用してはなりません。
  • 未成年の写真、性的・暴力的・差別的な内容を含む写真は入力しないでください。

第3条 生成画像の取り扱い

本アプリで生成される画像はフィクションであり、生物学的・遺伝的に正確な予測ではありません。生成画像を他者へ提示する際は、AI生成物であることを明示してください。生成画像の二次利用についてはユーザーの責任において行うものとします。

第4条 データの取り扱い

入力された写真は、生成処理のために Google Gemini API および（フォールバック時のみ）OpenAI API へ送信されます。本アプリ自身は写真を端末外に長期保存しません。各API事業者のプライバシーポリシーが適用されます。

第5条 免責

本アプリは現状有姿で提供されます。生成結果の品質、特定の用途への適合性、利用により生じた損害について、開発者は一切の責任を負いません。

第6条 規約の変更

本規約は予告なく変更される場合があります。最新の規約は常に本ページに掲載されます。
