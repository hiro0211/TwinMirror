要点 本アプリは、選択した2枚の顔写真を画像生成のために Google LLC の Gemini API へ送信します。送信前にユーザーから明示的な同意を取得し、当アプリの開発者サーバーには写真を保存しません。

1取得する情報

本アプリは、以下の情報のみを処理します。

  • 写真：ユーザーが選択した2枚の顔写真。生成処理にのみ使用します。
  • 性別・年齢の選択：生成リクエストのプロンプトに含まれます。
  • 同意ステータス／利用回数：端末内（UserDefaults）にのみ保存され、外部送信されません。

氏名、メールアドレス、位置情報、その他の個人情報は一切取得しません。

2利用目的

取得した写真は、子ども画像の生成リクエストを Google Gemini API へ送信するためにのみ使用されます。本アプリは初回生成前にこの送信について明示的な同意をユーザーから取得します。同意なしに写真が送信されることはありません。

3第三者提供（AI 処理）

本アプリは Apple App Store Review Guideline 5.1.2(i)（2025年11月施行）に従い、以下の第三者 AI サービスへの個人データ送信を明示的に開示します。

  • Google LLC（Gemini API） — 画像生成処理のため。Google プライバシーポリシー

本アプリの開発者自身が運営するサーバーには、写真データを保存しません。

4データの保存期間

選択された写真は、生成処理完了後、端末内のメモリから破棄されます。生成された子ども画像は、ユーザーが明示的に「保存」を選択した場合のみ、端末の写真ライブラリに保存されます。Google 側での画像の保持ポリシーは Google のプライバシーポリシーに従います。

5アクセス許可

本アプリは以下の権限を要求します。

  • 写真ライブラリの読み取り：入力写真を選択するため。
  • 写真ライブラリへの書き込み：生成結果を保存するため（任意）。

6トラッキング

本アプリはユーザーのトラッキングを行いません。App Tracking Transparency（ATT）の同意は要求しません。広告 SDK・アナリティクス SDK・クラッシュレポート SDK は組み込まれていません。

7利用回数の制限

本アプリはコスト管理のため、1日あたりの生成回数を端末ローカルで制限しています。この情報は端末内にのみ保存され、外部に送信されることはありません。

8子供のプライバシー

本アプリは18歳未満の方の利用を想定していません。また、未成年者の顔写真の入力は禁止しています。

9改定

本ポリシーは予告なく改定される場合があります。最新版は常に本ページに掲載されます。
